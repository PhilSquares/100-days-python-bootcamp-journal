# Day 17 Project — 🧩 Quiz Game (OOP Edition)

## Description
A command-line quiz application built using Object-Oriented Programming principles. The program presents a series of True/False questions, evaluates user input, and tracks performance throughout the session.

The project emphasizes modular design by separating responsibilities across multiple classes and files.

## Skills Practiced
- Object-Oriented Programming (OOP).
- Class design and instantiation.
- Encapsulation of data and behavior.
- Managing application state across objects.
- Working with multiple Python modules/files.
- Iterative control flow using loops.

## How to Run
```bash
python main.py
```

## Project Structure
- `main.py` → Entry point; initializes quiz and controls execution flow.
- `question_model.py` → Defines the Question class.
- `quiz_brain.py` → Contains quiz logic and state management.
- `data.py` → Stores question dataset.

