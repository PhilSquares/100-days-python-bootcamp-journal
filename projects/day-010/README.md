# Day 10 Project — Calculator

## 📖 Description
A command-line calculator that supports basic arithmetic operations:

- Addition (+)
- Subtraction (-)
- Multiplication (*)
- Division (/)

The program allows users to:
- Perform calculations
- Chain multiple operations using previous results
- Restart calculations at any time

---

## 🛠 Skills Practiced
- Writing functions with return values
- Using dictionaries to map operations to functions
- Control flow with loops and conditionals
- Function reuse and modular design
- Recursion for restarting program flow
- Handling user input and formatting output

---

## ▶️ How to Run
```bash
python main.py
```
