# Day 13 Project — Debugging Exercises

## 📖 Description
A set of exercises focused on identifying and fixing common programming errors in Python.

Each exercise highlights a different type of bug:
- Logic errors
- Syntax errors
- Runtime errors
- Incorrect assumptions in code behavior

---

## 🛠 Skills Practiced
- Debugging techniques and problem-solving
- Reading and interpreting error messages
- Tracing code execution
- Fixing logical and syntactical issues
- Understanding common pitfalls in Python

---

```text
exercises/

```

## 💡 Key Takeaway
- Debugging is a critical skill for any developer. These exercises reinforce the importance of understanding how code executes and how to systematically identify and fix issues.
