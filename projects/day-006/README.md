# Day 6 Project — Reeborg Maze Solver 🤖

## Description
This exercise focuses on using **Python functions and control flow** to solve a maze navigation problem.

Using the Reeborg robot simulator, a program was written to guide a robot through a maze by following the **right-hand rule algorithm**. The robot continuously checks if it can turn right, move forward, or must turn left in order to eventually reach the exit.

Although the challenge runs in an online simulator rather than the local terminal, the logic demonstrates how functions and conditionals can be used to solve real-world navigation and pathfinding problems.

## Skills Practiced
- Python function creation (`def`)
- Writing reusable helper functions
- Conditional logic (`if / elif / else`)
- While loops for continuous evaluation
- Algorithmic problem solving
- Code readability and modular design

## How to Run

This code is designed to run in the Reeborg Python simulator:

https://reeborg.ca/reeborg.html

Steps:
1. Open the simulator.
2. Select the **Maze** challenge.
3. Paste the code from `main.py`.
4. Run the program to watch Reeborg solve the maze.

Example command for local viewing:

```bash
python main.py
```

Note: The functions like `move()` and `turn_left()` are provided by the Reeborg simulator environment.
