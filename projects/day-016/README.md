# Day 16 Project — Coffee Machine (OOP Refactor) ☕️

## 📖 Description
An object-oriented refactor of the Coffee Machine project.  
This version restructures the application using classes to separate concerns and model real-world components more effectively.

The system is divided into:
- `Menu` → Handles available drinks
- `CoffeeMaker` → Manages resources and drink preparation
- `MoneyMachine` → Handles payment processing

---

## 🛠 Skills Practiced
- Object-Oriented Programming (OOP)
- Class design and instantiation
- Encapsulation (attributes + methods)
- Separation of concerns
- Refactoring procedural code into OOP architecture
- Multi-file project organization

---

## ▶️ How to Run
```bash
python main.py
```
