# Day 11 Project — Blackjack Capstone Project

## 📖 Description
A command-line version of the classic Blackjack game where the player competes against a computer dealer.

Game rules implemented:
- Try to get as close to 21 as possible without going over.
- Face cards (J, Q, K) are worth 10.
- Ace can be worth 11 or 1.
- Dealer must draw until reaching at least 17.

---

## 🛠 Skills Practiced
- Function design and modular programming
- Game logic implementation
- Control flow with loops and conditionals
- Working with lists to manage dynamic data
- Edge case handling (Blackjack, bust, Ace adjustment)
- Code organization and readability
