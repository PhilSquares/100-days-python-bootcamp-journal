# Day 9 Project — Secret Auction 🏆

## Description
A command-line auction system that allows multiple users to place bids anonymously.

Each participant enters their name and bid amount, which are stored in a dictionary. Once all bids are submitted, the program determines and displays the highest bidder.

This project introduces dictionary-based data storage and demonstrates how to process collections of data to compute results.

## Skills Practiced
- Python dictionaries (key-value pairs)
- Iterating through dictionaries
- Function creation and separation of concerns
- State management across loops
- User input handling
- Basic algorithm design (finding maximum value)

## How to Run
```bash
python main.py
```

Example Flow:

```python
What is your name?: Alice
What is your bid?: $100
Are there any other bidders? yes

What is your name?: Bob
What is your bid?: $150
Are there any other bidders? no

The winner is Bob with a bid of $150
```
