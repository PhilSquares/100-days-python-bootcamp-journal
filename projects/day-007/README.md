# Day 7 Project — Hangman 🧍‍♂️

## Description
A command-line implementation of the classic Hangman game built using Python.

The program randomly selects a word and prompts the user to guess letters one at a time. It tracks correct guesses, remaining lives, and displays progress after each attempt. The game ends when the user either successfully guesses the word or runs out of lives.

This project focuses on managing state, control flow, and user interaction within a loop-driven program.

## Skills Practiced
- Python loops (`while`, `for`)
- Conditional logic (`if / elif / else`)
- State management using variables and lists
- String construction and dynamic output
- User input handling
- Basic game development logic

## How to Run
```bash
python main.py
```
