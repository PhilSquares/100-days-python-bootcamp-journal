# Day 8 Project — Caesar Cipher 🔐

## Description
A command-line tool that implements the Caesar Cipher encryption technique.

Users can encode or decode messages by shifting letters forward or backward in the alphabet by a specified number. The program preserves spaces and special characters, making it more flexible for real-world input.

This project demonstrates how functions and parameters can be used to create reusable and scalable logic.

## Skills Practiced
- Python functions with parameters
- Conditional logic for multi-behavior functions
- String manipulation and transformation
- Modular arithmetic (`%`) for wrapping values
- Looping for continuous program execution
- Handling edge cases in user input

## How to Run
```bash
python main.py
```
