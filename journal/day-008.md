## 💻 Project of the Day
- **Name:** Caesar Cipher Tool
- **Description:**  
  Built a command-line encryption tool that uses the Caesar Cipher technique to encode and decode messages. The program shifts letters based on a user-defined value while preserving non-alphabet characters.

## 🤔 Challenges & Solutions
- **Challenge:** Handling shifts that go beyond the alphabet range.
- **Solution:** Used the modulo operator (`%`) to wrap around the alphabet list.

- **Challenge:** Supporting both encoding and decoding in a single function.
- **Solution:** Adjusted the shift direction dynamically by multiplying by `-1` when decoding.

- **Challenge:** Preserving spaces and special characters.
- **Solution:** Added a conditional check to leave non-alphabet characters unchanged.
