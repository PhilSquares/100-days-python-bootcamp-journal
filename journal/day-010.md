## 💻 Project of the Day
- **Name:** Calculator
- **Description:**  
A command-line calculator that allows users to perform continuous mathematical operations.  
Users can chain calculations together (e.g., use the result of a previous operation as the next input) or restart the calculator at any time.

---

## 🤔 Challenges & Solutions

- **Challenge:** Understanding how to dynamically call functions based on user input  
  - **Solution:** Used a dictionary to map operation symbols (`+`, `-`, `*`, `/`) to their corresponding functions.

- **Challenge:** Keeping the calculator running for multiple operations  
  - **Solution:** Implemented a `while` loop with a flag (`should_accumulate`) to control execution flow.

- **Challenge:** Restarting the calculator cleanly  
  - **Solution:** Used recursion by calling `calculator()` again when the user chooses to start over.
