## 💻 Project of the Day
- **Name:** Coffee Machine (OOP Refactor)
- **Description:**  
A refactored version of the Coffee Machine project using Object-Oriented Programming. The system is now broken into multiple classes (`Menu`, `CoffeeMaker`, `MoneyMachine`) to better organize responsibilities and improve code maintainability.

## 🤔 Challenges & Solutions
- **Challenge:** Understanding how classes interact with each other  
  **Solution:** Identified clear responsibilities for each class and used method calls to coordinate behavior.

- **Challenge:** Transitioning from procedural to object-oriented thinking  
  **Solution:** Mapped previous functions into class methods and grouped related data with behavior.

- **Challenge:** Managing state across multiple objects  
  **Solution:** Leveraged class attributes (e.g., resources, profit) to maintain state within each object.
