## 💻 Project of the Day
- **Name:** 🧩 Quiz Game (OOP Edition)
- **Description:** Built a command-line quiz application that presents True/False questions to the user, evaluates responses, and tracks score using a modular, object-oriented design.

## 🤔 Challenges & Solutions
- **Challenge:** Understanding how different classes interact with each other.
  **Solution:** Broke down responsibilities:
  - `Question` → stores question data
  - `QuizBrain` → controls game logic and flow

- **Challenge:** Managing state across multiple questions.  
  **Solution:** Used class attributes (`score`, `question_number`) to persist state throughout the quiz.
