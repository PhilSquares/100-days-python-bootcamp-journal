## 💻 Project of the Day
- **Name:** Secret Auction
- **Description:**  
  Built a command-line auction program where multiple users can submit bids. The program stores each bidder’s name and bid amount in a dictionary, then determines the highest bidder once all entries are collected.

## 🤔 Challenges & Solutions
- **Challenge:** Storing multiple bidders and their associated bids.
- **Solution:** Used a dictionary (`bids`) to map bidder names to bid amounts.

- **Challenge:** Determining the highest bidder efficiently.
- **Solution:** Created a `find_highest_bidder()` function that iterates through the dictionary and tracks the maximum value.

- **Challenge:** Allowing multiple users to input data without interference.
- **Solution:** Used a loop with a continuation flag and cleared the screen between entries using newline spacing.
