## 🤔 Challenges & Solutions

- **Challenge:** Loop never reaching expected condition  
  - **Solution:** Adjusted range to include correct boundary (`range(1, 21)`).

- **Challenge:** Index out of range error  
  - **Solution:** Fixed indexing mismatch between list and random number generation.

- **Challenge:** Incorrect conditional logic  
  - **Solution:** Adjusted comparison operators to properly categorize age groups.

- **Challenge:** Syntax and indentation issues  
  - **Solution:** Corrected indentation and string formatting.

- **Challenge:** Variable not updating correctly  
  - **Solution:** Fixed assignment vs comparison (`=` vs `==`).
