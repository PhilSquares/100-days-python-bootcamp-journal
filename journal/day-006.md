## 💻 Project of the Day
- **Name:** Reeborg Maze Solver
- **Description:**  
  Used Python functions and conditional logic to guide a robot through a maze using the **right-hand rule** algorithm. The robot checks its surroundings and decides whether to turn right, move forward, or turn left until it reaches the goal.
Link to web version of the challenge: [Reeborg's World](https://reeborg.ca/reeborg.html?lang=en&mode=python&menu=worlds%2Fmenus%2Freeborg_intro_en.json&name=Maze&url=worlds%2Ftutorial_en%2Fmaze1.json)

## 🤔 Challenges & Solutions
- **Challenge:** Understanding how the robot determines which direction to move in a dynamic environment.
- **Solution:** Implemented conditional checks (`right_is_clear()`, `front_is_clear()`) and used a loop to continuously evaluate the environment until the goal was reached.
  
- **Challenge:** Avoiding repetitive code for turning right.
- **Solution:** Created a reusable `turn_right()` helper function built from the existing `turn_left()` function.
