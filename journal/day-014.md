## 💻 Project of the Day
- **Name:** Higher/Lower Game
- **Description:**  
A command-line game where the user is presented with two public figures or brands and must guess which one has more social media followers. The game continues as long as the user guesses correctly, with the score increasing after each correct answer.

## 🤔 Challenges & Solutions
- **Challenge:** Preventing duplicate comparisons (same account appearing twice).  
  **Solution:** Added a check to ensure `account_a` and `account_b` are not the same.
  
- **Challenge:** Managing game state across iterations.  
  **Solution:** Used a `while` loop with a `game_should_continue` flag and tracked score separately.

- **Challenge:** Keeping code readable as complexity increased.  
  **Solution:** Broke logic into helper functions like `format_data()` and `check_answer()`.
