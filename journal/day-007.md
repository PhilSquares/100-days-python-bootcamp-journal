## 💻 Project of the Day
- **Name:** Hangman Game
- **Description:**  
  Built a command-line version of the classic Hangman game. The program randomly selects a word and allows the user to guess letters while tracking remaining lives. The game dynamically updates the displayed word and ends when the player either guesses correctly or runs out of lives.

## 🤔 Challenges & Solutions
- **Challenge:** Keeping track of correct guesses without losing previous progress.
- **Solution:** Used a `correct_letters` list to store successful guesses and rebuild the display string on each iteration.

- **Challenge:** Preventing repeated guesses from affecting game logic.
- **Solution:** Added a check to notify the user if a letter was already guessed.

- **Challenge:** Managing game state (win/lose conditions).
- **Solution:** Used a `game_over` flag and conditional checks to control when the loop exits.
