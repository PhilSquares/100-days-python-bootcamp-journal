## 💻 Project of the Day
- **Name:** Number Guessing Game
- **Description:**  
A command-line game where the user tries to guess a randomly generated number between 1 and 100.  
The player selects a difficulty level, which determines the number of attempts allowed.

---

## 🤔 Challenges & Solutions

- **Challenge:** Managing the number of attempts remaining.  
  - **Solution:** Passed `turns` into the `check_answer()` function and returned the updated value.

- **Challenge:** Structuring game flow cleanly.  
  - **Solution:** Encapsulated the main logic inside a `game()` function.

- **Challenge:** Handling difficulty levels dynamically.  
  - **Solution:** Created a `set_difficulty()` function to return different attempt values.

- **Challenge:** Preventing infinite loops.  
  - **Solution:** Used a `while` loop with clear exit conditions (correct guess or no turns left).
