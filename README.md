## 🛠 How to Use This Repo

### Starting a New Day
1. **Journal Entry**  
   Copy `templates/journal-entry-template.md` → `journal/day-XXX.md`  
   Fill in:
   - What you learned.
   - Project details.
   - Challenges faced.
   - Next steps.

2. **Project Folder**  
   Create a folder in `projects/day-XXX`  
   Copy `templates/project-readme-template.md` into it and document your project.  
   Add your code in the same folder.

3. **Progress Tracker**  
   Update the table below with:
   - Day number
   - Date
   - Project name
   - Link to your code

4. **Commit Changes**  
git add .
git commit -m "feat: add day-XXX project and journal entry"
git push
